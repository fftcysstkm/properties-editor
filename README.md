# Properties Editor

Unicodeエスケープされた `.properties` ファイルを日本語で表示・編集・検索できるVS Code拡張機能です。

## 特徴

*   **日本語表示**: `\uXXXX` 形式のUnicodeエスケープを自動的にデコードして日本語で表示します。
*   **日本語編集**: 日本語で入力・編集して保存すると、自動的にUnicodeエスケープ形式に変換して保存されます。
*   **検索機能**: エディタ内で `Ctrl+F` (Mac: `Cmd+F`) を押すと、日本語で検索が可能です。

## 利用方法

1.  `.properties` ファイルを開きます。
2.  自動的に専用のエディタが表示され、日本語で内容を確認できます。
3.  通常通り編集し、`Ctrl+S` (Mac: `Cmd+S`) で保存します。

## インストール方法

### 開発環境での実行 (デバッグ)
1.  このプロジェクトをVS Codeで開きます。
2.  `F5` キーを押してデバッグを開始します。
3.  新しいウィンドウが開くので、そこで `.properties` ファイルを開いて動作を確認します。

### パッケージからのインストール
1.  `properties-editor-0.0.1.vsix` ファイルを作成します（`npx vsce package`）。
2.  VS Codeの拡張機能ビューを開きます。
3.  右上の「...」メニューから「VSIXからのインストール...」を選択し、作成した `.vsix` ファイルを選択します。
