# Change Log

All notable changes to the "unicode-properties-editor" extension will be documented in this file.

## [0.0.5] - 2025-12-04

- Fixed a race condition that caused the editor to remain blank when switching tabs.

## [0.0.4] - 2025-12-04

- Fixed an issue where the editor content would disappear when switching tabs.
- Updated documentation regarding potential conflicts with other extensions.

## [0.0.3] - 2025-12-04

- Fixed an issue where the cursor would jump to the end of the file during input.

## [0.0.2] - 2025-12-03

- Renamed extension to `unicode-properties-editor`.
- Updated description to be language-neutral.

## [0.0.1] - 2025-12-03

- Initial release
- Support for displaying, editing, and searching Unicode escaped properties files in Japanese and other languages.
